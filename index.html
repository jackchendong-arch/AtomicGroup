Workflow

Recruiter Checkpoints

The product keeps a visible, human-controlled path from raw documents to hiring-manager handoff.

1. Import CV and JD Load both source documents from the local machine.
2. Review Extracted Evidence Check that parsing captured the right signals before generation.
3. Refine Candidate Narrative Edit the generated profile before it can be approved.
4. Prepare Manager Handoff Move the approved summary into the email workflow.
